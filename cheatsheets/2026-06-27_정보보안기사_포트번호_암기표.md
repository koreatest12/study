# 정보보안기사 포트번호 암기표

> 정보보안기사 네트워크 보안 과목에서 자주 나오는 포트 번호를 빠르게 암기하기 위한 자료입니다.

---

## 필수 암기 포트

| 서비스 | 포트 | 프로토콜 | 암기 포인트 |
|---|---:|---|---|
| FTP Data | 20 | TCP | FTP 데이터 전송 |
| FTP Control | 21 | TCP | FTP 제어 연결 |
| SSH | 22 | TCP | 보안 원격 접속 |
| Telnet | 23 | TCP | 평문 원격 접속, 보안 취약 |
| SMTP | 25 | TCP | 메일 송신 |
| DNS | 53 | UDP/TCP | 질의는 주로 UDP, Zone Transfer는 TCP |
| DHCP Server | 67 | UDP | 서버 포트 |
| DHCP Client | 68 | UDP | 클라이언트 포트 |
| TFTP | 69 | UDP | 단순 파일 전송 |
| HTTP | 80 | TCP | 웹 평문 통신 |
| POP3 | 110 | TCP | 메일 수신 |
| NTP | 123 | UDP | 시간 동기화 |
| NetBIOS | 137~139 | TCP/UDP | 윈도우 파일 공유 관련 |
| IMAP | 143 | TCP | 메일 수신/동기화 |
| SNMP | 161 | UDP | 네트워크 장비 관리 |
| SNMP Trap | 162 | UDP | SNMP 알림 |
| LDAP | 389 | TCP/UDP | 디렉터리 서비스 |
| HTTPS | 443 | TCP | TLS 기반 웹 통신 |
| SMB/CIFS | 445 | TCP | 윈도우 파일 공유 |
| IKE | 500 | UDP | IPsec 키 교환 |
| LDAPS | 636 | TCP | TLS 기반 LDAP |
| RDP | 3389 | TCP | 윈도우 원격 데스크톱 |
| NAT-T | 4500 | UDP | NAT 환경 IPsec |

---

## 시험에 자주 나오는 구분

### 보안 프로토콜

| 항목 | 포트 | 설명 |
|---|---:|---|
| SSH | 22 | Telnet 대체 보안 원격 접속 |
| HTTPS | 443 | HTTP + TLS |
| LDAPS | 636 | LDAP + TLS |
| IPsec IKE | 500 | 키 교환 |
| IPsec NAT-T | 4500 | NAT 환경 터널링 |

### 취약하게 자주 언급되는 평문 프로토콜

| 항목 | 포트 | 취약점 |
|---|---:|---|
| FTP | 20/21 | 계정/비밀번호 평문 전송 가능 |
| Telnet | 23 | 원격 접속 정보 평문 전송 |
| HTTP | 80 | 암호화 없음 |
| POP3 | 110 | 기본 POP3는 평문 가능 |
| SNMP v1/v2 | 161 | community string 취약 가능 |

---

## 빠른 암기 문장

- `22 SSH`는 보안 원격 접속이다.
- `23 Telnet`은 평문 원격 접속이라 위험하다.
- `53 DNS`는 UDP가 기본이고 TCP도 사용한다.
- `80 HTTP`, `443 HTTPS`는 웹 보안 문제의 기본이다.
- `500 IKE`, `4500 NAT-T`는 IPsec 문제에서 자주 나온다.
- `3389 RDP`는 윈도우 원격 접속이다.
