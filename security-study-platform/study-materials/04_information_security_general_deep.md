# 4과목 정보보안 일반 심화 정리

## 1. 핵심 범위

정보보안 일반은 CIA, AAA, 암호학, 해시, 전자서명, PKI, 접근통제, 보안 모델을 다룬다.

---

## 2. CIA와 AAA

| 구분 | 요소 | 설명 |
|---|---|---|
| CIA | 기밀성 | 허가된 사용자만 접근 |
| CIA | 무결성 | 데이터가 임의 변경되지 않음 |
| CIA | 가용성 | 필요한 시점에 사용 가능 |
| AAA | 인증 | 사용자가 누구인지 확인 |
| AAA | 인가 | 무엇을 할 수 있는지 권한 확인 |
| AAA | 책임추적성 | 행위 추적과 감사 가능 |

---

## 3. 암호 기술

| 구분 | 예시 | 특징 |
|---|---|---|
| 대칭키 | AES, SEED, ARIA | 빠름, 키 분배 문제 |
| 공개키 | RSA, ECC | 느림, 키 교환/전자서명 활용 |
| 해시 | SHA-256, SHA-3 | 무결성 검증, 복호화 불가 |
| 전자서명 | RSA 서명, ECDSA | 인증, 무결성, 부인방지 |

---

## 4. 접근통제

| 모델 | 설명 |
|---|---|
| DAC | 소유자가 접근 권한을 결정 |
| MAC | 보안 등급과 정책에 따라 강제 통제 |
| RBAC | 역할에 따라 권한 부여 |

---

## 5. 보안 모델

| 모델 | 핵심 |
|---|---|
| Bell-LaPadula | 기밀성, No Read Up / No Write Down |
| Biba | 무결성, No Read Down / No Write Up |
| Clark-Wilson | 상업적 무결성, 직무 분리 |

---

## 6. 예상문제

### 문제 1
해시 함수의 설명으로 옳지 않은 것은?

1. 고정 길이 출력값을 생성한다.
2. 무결성 검증에 사용된다.
3. 복호화가 가능하다.
4. 충돌 저항성이 중요하다.

**정답: 3**

**해설:** 해시는 일방향 함수로 복호화 개념이 없다.

### 문제 2
RBAC의 기준은?

1. 역할
2. 파일 소유자
3. 보안 등급
4. 포트 번호

**정답: 1**

**해설:** RBAC는 Role-Based Access Control로 역할 기반 접근통제다.
