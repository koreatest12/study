# 홈페이지 구조도

## 1. 메인 화면 구성

```text
┌──────────────────────────────────────────────┐
│ Hero 영역                                     │
│ - 정보보안기사 학습 플랫폼 제목               │
│ - 랜덤 문제 풀기 버튼                         │
│ - 오답노트 보기 버튼                          │
└──────────────────────────────────────────────┘

┌──────────────────────────────────────────────┐
│ 과목 카드 영역                                │
│ [시스템] [네트워크] [어플리케이션] [일반] [법규] │
└──────────────────────────────────────────────┘

┌──────────────────────────────────────────────┐
│ 문제풀이 영역                                 │
│ - 문제 제목                                   │
│ - 4지선다 선택지                              │
│ - 정답/오답 표시                              │
│ - 해설 출력                                   │
└──────────────────────────────────────────────┘

┌──────────────────────────────────────────────┐
│ 오답노트 영역                                 │
│ - 틀린 문제 저장                              │
│ - 선택한 답                                   │
│ - 정답                                       │
│ - 해설                                       │
└──────────────────────────────────────────────┘
```

---

## 2. 사용자 흐름

```text
1. 사용자가 홈페이지 접속
2. 랜덤 문제 또는 과목 카드 클릭
3. 서버에서 문제 JSON 반환
4. 사용자가 선택지 클릭
5. 정답 여부와 해설 표시
6. 오답이면 localStorage에 저장
7. 오답노트에서 재확인
```

---

## 3. 화면별 역할

| 영역 | 역할 |
|---|---|
| Hero | 학습 플랫폼 소개와 시작 버튼 제공 |
| Subject Cards | 과목별 랜덤 문제 진입 |
| Quiz Panel | 문제풀이와 정답 해설 제공 |
| Wrong Note | 틀린 문제 복습 |
| Footer | 저장소/프로젝트 설명 |

---

## 4. 반응형 구조

| 화면 | 동작 |
|---|---|
| PC | 카드가 여러 열로 표시 |
| 태블릿 | 카드가 2~3열로 표시 |
| 모바일 | 카드가 1열 중심으로 표시 |
