# 네트워크 보안 심화정리

---

## 1. OSI 7계층 핵심

| 계층 | 이름 | 주요 내용 |
|---:|---|---|
| 7 | 응용 | 웹, 메일, 파일 전송, 이름 해석 |
| 6 | 표현 | 암호화, 압축, 인코딩 |
| 5 | 세션 | 세션 연결과 종료 관리 |
| 4 | 전송 | TCP, UDP |
| 3 | 네트워크 | IP, ICMP, 라우팅 |
| 2 | 데이터링크 | Ethernet, ARP, 스위칭 |
| 1 | 물리 | 케이블, 신호, 허브 |

---

## 2. TCP와 UDP

| 구분 | TCP | UDP |
|---|---|---|
| 연결 방식 | 연결형 | 비연결형 |
| 신뢰성 | 높음 | 낮음 |
| 속도 | 상대적으로 느림 | 빠름 |
| 대표 예시 | 웹 보안 통신, 원격 접속 | 이름 해석, 시간 동기화 |

---

## 3. 보안 장비 비교

| 장비 | 역할 |
|---|---|
| 방화벽 | 접근 허용과 차단 |
| IDS | 이상 행위 탐지와 경고 |
| IPS | 이상 행위 탐지와 차단 |
| WAF | 웹 계층 보안 보호 |
| NAC | 네트워크 접근 제어 |
| VPN | 암호화된 통신 터널 제공 |

---

## 4. 주요 포트 복습

| 서비스 | 포트 |
|---|---:|
| SSH | 22 |
| DNS | 53 |
| HTTP | 80 |
| HTTPS | 443 |
| IKE | 500 |
| NAT-T | 4500 |
| RDP | 3389 |

---

## 5. 시험 포인트

- IDS는 탐지 중심이다.
- IPS는 탐지 후 차단까지 수행한다.
- VPN은 터널링과 암호화를 제공한다.
- WAF는 웹 계층 보호에 특화되어 있다.
- TCP 연결 수립은 SYN, SYN/ACK, ACK 순서이다.
- HTTP는 80, HTTPS는 443을 기본 포트로 사용한다.

## 6. 예상문제

### 문제 1
TCP와 UDP의 차이로 적절한 것은?

1. TCP는 비연결형, UDP는 연결형이다.
2. TCP는 신뢰성 중심, UDP는 속도 중심이다.
3. 둘 다 전송 계층이 아니다.
4. UDP는 항상 3단계 연결 수립을 수행한다.

정답: 2

해설: TCP는 연결형과 신뢰성 중심이고, UDP는 비연결형과 속도 중심이다.

### 문제 2
웹 계층 보호에 특화된 장비는?

1. WAF
2. Hub
3. Repeater
4. Modem

정답: 1

해설: WAF는 웹 요청과 응답을 기준으로 웹 계층 보안 위협을 완화한다.
