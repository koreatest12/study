### 📚 학습 커리큘럼 (상세 보기)
<details>
<summary>👆 **1. 시스템 보안 (System Security) - 클릭하여 펼치기**</summary>

- **운영체제 구조:** CPU/메모리 관리, 프로세스 스케줄링, 교착상태(Deadlock)
- **클라이언트 보안:** 윈도우 레지스트리, 악성코드(바이러스/랜섬웨어) 분석 및 대응
- **서버 보안:** 계정 관리(/etc/passwd), 권한 관리(chmod), 서비스 보안
- **로그 분석:** Syslog, wtmp/utmp, Windows Event Log, 침해사고 대응
</details>

<details>
<summary>👆 **2. 네트워크 보안 (Network Security)**</summary>

- **네트워크 일반:** OSI 7계층, TCP/IP 헤더 구조
- **공격 기법:** DoS/DDoS, Spoofing, Sniffing, Session Hijacking
- **보안 장비:** 방화벽, IDS/IPS, VPN, NAC
- **무선 보안:** WEP, WPA/WPA2, 802.1x
</details>

<details>
<summary>👆 **3. 어플리케이션 보안 (App Security)**</summary>

- **웹 보안:** OWASP Top 10 (SQL Injection, XSS, CSRF)
- **DB 보안:** 접근통제, 무결성, 암호화
- **전자상거래:** SET, SSL/TLS 프로토콜
</details>

<details>
<summary>👆 **4. 정보보안 일반 (General)**</summary>

- **보안 개요:** 기밀성, 무결성, 가용성 (CIA)
- **암호학:** 대칭키/공개키, 해시함수, 전자서명, PKI
- **접근 통제:** MAC, DAC, RBAC, 식별/인증/인가
</details>

<details>
<summary>👆 **5. 관리 및 법규 (Compliance)**</summary>

- **관리체계:** ISMS-P, 위험 관리(Risk Management), BCP/DRP
- **법규:** 정보통신망법, 개인정보보호법, 정보통신기반보호법
</details>
