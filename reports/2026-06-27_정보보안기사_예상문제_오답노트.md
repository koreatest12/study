## 1과목. 시스템 보안 예상문제

### 문제 1
리눅스에서 `/etc/shadow` 파일의 주요 목적은?

1. 사용자 홈 디렉터리 저장  
2. 암호화된 패스워드 해시 및 계정 만료 정보 저장  
3. 네트워크 라우팅 정보 저장  
4. 프로세스 실행 로그 저장

**정답: 2**

**해설:** `/etc/shadow`는 암호화된 패스워드 해시, 계정 만료일, 패스워드 변경 주기 등을 저장한다. 일반 사용자에게 노출되면 안 된다.

### 문제 2
리눅스에서 새 파일 생성 시 기본 권한을 제한하는 값은?

1. chmod  
2. chown  
3. umask  
4. sudo

**정답: 3**

**해설:** `umask`는 새로 생성되는 파일과 디렉터리의 기본 권한에서 제거할 권한을 지정한다.

### 문제 3
윈도우에서 NTFS 권한에 해당하는 것은?

1. 읽기  
2. 쓰기  
3. 수정  
4. 모두 해당

**정답: 4**

**해설:** NTFS 권한에는 읽기, 쓰기, 수정, 모든 권한 등이 포함된다.

## 2과목. 네트워크 보안 예상문제

### 문제 1
HTTP와 HTTPS의 기본 포트 조합으로 올바른 것은?

1. HTTP 22 / HTTPS 23  
2. HTTP 53 / HTTPS 110  
3. HTTP 80 / HTTPS 443  
4. HTTP 500 / HTTPS 4500

**정답: 3**

**해설:** HTTP는 TCP 80, HTTPS는 TCP 443을 기본 포트로 사용한다.

### 문제 2
ARP Spoofing 공격의 주요 목적은?

1. MAC 주소와 IP 주소 매핑을 속여 중간자 공격 수행  
2. 웹 서버의 SQL 구문 조작  
3. 파일 권한 상승  
4. 패스워드 복잡도 검사

**정답: 1**

**해설:** ARP Spoofing은 공격자가 자신의 MAC 주소를 게이트웨이나 특정 호스트의 MAC 주소처럼 속여 트래픽을 가로채는 공격이다.

### 문제 3
Stateful Firewall의 특징은?

1. 패킷의 상태 정보를 고려하지 않는다.  
2. 연결 상태를 추적하여 허용 여부를 판단한다.  
3. 물리 계층 신호만 증폭한다.  
4. 악성코드 파일만 검사한다.

**정답: 2**

**해설:** Stateful Firewall은 세션과 연결 상태를 추적하여 패킷 허용 여부를 판단한다.

## 3과목. 어플리케이션 보안 예상문제

### 문제 1
XSS 공격 방어에 가장 중요한 보안 조치는?

1. 출력값 인코딩  
2. 관리자 계정 공유  
3. 오류 메시지 상세 출력  
4. 파일 실행 권한 부여

**정답: 1**

**해설:** XSS는 악성 스크립트가 브라우저에서 실행되는 공격이므로 출력값 인코딩이 핵심 방어 방법이다.

### 문제 2
CSRF 공격의 특징으로 적절한 것은?

1. 사용자의 인증 상태를 악용해 원치 않는 요청을 전송하게 한다.  
2. 데이터베이스 쿼리 구조를 변경한다.  
3. 포트 스캔을 수행한다.  
4. 파일 시스템 권한을 변경한다.

**정답: 1**

**해설:** CSRF는 사용자가 로그인한 상태를 악용해 공격자가 의도한 요청을 사용자의 권한으로 실행하게 한다.

### 문제 3
파일 업로드 취약점 대응으로 적절하지 않은 것은?

1. 확장자 검증  
2. MIME 타입 검증  
3. 업로드 디렉터리 실행 권한 제거  
4. 업로드 파일을 웹 루트에서 바로 실행 가능하게 설정

**정답: 4**

**해설:** 업로드 파일이 웹 루트에서 바로 실행되면 웹쉘 공격으로 이어질 수 있다.

## 4과목. 정보보안 일반 예상문제

### 문제 1
대칭키 암호 알고리즘에 해당하는 것은?

1. AES  
2. RSA  
3. ECC  
4. DSA

**정답: 1**

**해설:** AES는 대표적인 대칭키 암호 알고리즘이다. RSA와 ECC는 공개키 암호 방식이다.

### 문제 2
전자서명이 제공하는 보안 속성으로 가장 적절한 것은?

1. 무결성, 인증, 부인방지  
2. 가용성, 압축, 속도 향상  
3. 은닉성, 삭제, 복구  
4. 라우팅, 스위칭, 포워딩

**정답: 1**

**해설:** 전자서명은 송신자 인증, 메시지 무결성, 부인방지를 제공한다.

### 문제 3
Biba 모델의 핵심 목적은?

1. 기밀성 보호  
2. 무결성 보호  
3. 가용성 향상  
4. 장애 복구

**정답: 2**

**해설:** Biba 모델은 무결성 보호를 목적으로 하는 보안 모델이다.

## 5과목. 정보보안 관리 및 법규 예상문제

### 문제 1
BCP의 의미로 가장 적절한 것은?

1. 업무 연속성 계획  
2. 백업 압축 정책  
3. 브라우저 캐시 보호  
4. 네트워크 포트 차단

**정답: 1**

**해설:** BCP는 Business Continuity Plan으로, 재해나 장애 상황에서도 핵심 업무를 지속하기 위한 계획이다.

### 문제 2
RTO의 의미는?

1. 복구 목표 시간  
2. 복구 목표 시점  
3. 암호 키 길이  
4. 네트워크 전송률

**정답: 1**

**해설:** RTO는 Recovery Time Objective로, 장애 발생 후 서비스를 복구해야 하는 목표 시간을 의미한다. RPO는 Recovery Point Objective로 복구 목표 시점이다.

### 문제 3
개인정보 처리 원칙으로 적절하지 않은 것은?

1. 목적에 필요한 최소한의 개인정보 수집  
2. 목적 달성 후 지체 없이 파기  
3. 정보주체 동의 없이 무제한 제3자 제공  
4. 안전성 확보 조치 수행

**정답: 3**

**해설:** 개인정보는 원칙적으로 동의 또는 법적 근거 없이 제3자에게 제공할 수 없다.

# 오답 정리 규칙

1. 틀린 문제는 문제 전체를 다시 쓰지 말고 핵심 문장만 기록한다.
2. 틀린 이유는 `개념 미숙`, `암기 오류`, `문제 해석 오류`, `실수` 중 하나로 분류한다.
3. 정답 근거는 한 줄로 작성한다.
4. 1일 후, 3일 후, 7일 후 같은 문제를 다시 푼다.
5. 같은 개념을 2회 이상 틀리면 최종 암기표에 추가한다.
