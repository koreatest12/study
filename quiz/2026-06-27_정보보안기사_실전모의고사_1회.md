# 정보보안기사 실전 모의고사 1회

> 구성: 과목별 5문제, 총 25문제  
> 목표: 기본 개념 점검과 오답노트 작성

---

## 1과목 시스템 보안

### 1번
리눅스에서 공용 디렉터리 파일 삭제를 제한하는 권한은?

1. SetUID
2. SetGID
3. Sticky Bit
4. sudo

정답: 3

해설: Sticky Bit는 공용 디렉터리에서 파일 소유자 또는 관리자만 삭제할 수 있게 한다.

### 2번
패스워드 해시와 계정 만료 정보를 저장하는 파일은?

1. /etc/passwd
2. /etc/shadow
3. /etc/hosts
4. /etc/services

정답: 2

해설: /etc/shadow에는 패스워드 해시와 만료 정책이 저장된다.

### 3번
업무 수행에 필요한 최소 권한만 부여하는 원칙은?

1. 권한 집중 원칙
2. 최소 권한 원칙
3. 관리자 우선 원칙
4. 공개 접근 원칙

정답: 2

해설: 최소 권한 원칙은 침해 사고 피해 범위를 줄이기 위한 기본 원칙이다.

### 4번
마지막 백업 이후 변경분만 백업하는 방식은?

1. 전체 백업
2. 증분 백업
3. 미러링
4. 콜드 백업

정답: 2

해설: 증분 백업은 마지막 백업 이후 변경된 데이터만 백업한다.

### 5번
윈도우 보안 로그 확인 도구는?

1. Event Viewer
2. cron
3. chmod
4. iptables

정답: 1

해설: Windows Event Viewer에서 보안, 시스템, 응용 프로그램 로그를 확인한다.

## 2과목 네트워크 보안

### 6번
HTTP 기본 포트는?

1. 22
2. 53
3. 80
4. 443

정답: 3

해설: HTTP는 TCP 80을 사용한다.

### 7번
HTTPS 기본 포트는?

1. 21
2. 23
3. 110
4. 443

정답: 4

해설: HTTPS는 TCP 443을 사용한다.

### 8번
IDS와 IPS의 차이로 적절한 것은?

1. IDS는 차단, IPS는 탐지만 수행한다.
2. IDS는 탐지 중심, IPS는 탐지와 차단을 수행한다.
3. 둘 다 물리 계층 장비이다.
4. 둘 다 백업 장비이다.

정답: 2

해설: IDS는 탐지와 경고, IPS는 탐지 후 차단까지 수행한다.

### 9번
IPsec NAT-T의 포트는?

1. UDP 500
2. UDP 4500
3. TCP 80
4. TCP 443

정답: 2

해설: NAT-T는 UDP 4500을 사용한다.

### 10번
TCP 연결 수립 순서는?

1. SYN, SYN/ACK, ACK
2. ACK, SYN, FIN
3. FIN, ACK, SYN
4. SYN, FIN, ACK

정답: 1

해설: TCP 3-way handshake는 SYN, SYN/ACK, ACK 순서이다.

## 3과목 어플리케이션 보안

### 11번
SQL Injection의 대표 대응은?

1. Prepared Statement
2. 오류 상세 출력
3. 관리자 계정 사용
4. 평문 쿠키 저장

정답: 1

해설: Prepared Statement는 SQL 구조와 입력값을 분리한다.

### 12번
XSS 방어의 핵심은?

1. 출력값 인코딩
2. 포트 차단
3. 디스크 백업
4. 라우팅 변경

정답: 1

해설: XSS는 스크립트 실행 공격이므로 출력값 인코딩이 중요하다.

### 13번
CSRF의 특징은?

1. SQL 구문 조작
2. 사용자의 인증 상태 악용
3. MAC 주소 변조
4. 파일 시스템 포맷

정답: 2

해설: CSRF는 로그인된 사용자의 권한으로 원치 않는 요청을 수행하게 한다.

### 14번
파일 업로드 취약점 대응으로 부적절한 것은?

1. 확장자 검증
2. MIME 검증
3. 실행 권한 제거
4. 웹 루트에서 바로 실행 허용

정답: 4

해설: 업로드 파일을 바로 실행 가능하게 하면 웹쉘 공격으로 이어질 수 있다.

### 15번
로그인 후 세션 ID를 재발급하는 이유는?

1. 세션 고정 공격 방지
2. DDoS 방지
3. ARP 공격 방지
4. 포트 스캔 방지

정답: 1

해설: 세션 고정 공격을 막기 위해 로그인 후 세션 ID를 새로 발급한다.
